##Mouse Actions (Negative Numbers)

--These codes simulate mouse inputs.
Code	Action	                Description
-1	    Left Mouse Button	        -The primary mouse click. Gains double-click if enabled.
-2	    Right Mouse Button	        -The secondary mouse click.
-3	    Middle Mouse Button	        -The scroll wheel click.
-4	    Mouse Button 4 (X1)	        -The "Back" side button on many mice.
-5	    Mouse Button 5 (X2)	        -The "Forward" side button on many mice.
-6	    Scroll Wheel Up	            -Simulates one "tick" of scrolling up. Is a single event, not held.
-7	    Scroll Wheel Down	        -Simulates one "tick" of scrolling down. Is a single event, not held.
